### Booking & Tech Setup

- **Booking platform:** Vagaro (already chosen). Vagaro offers an embeddable booking widget that can be placed on any external website — so the site can look fully custom while still using Vagaro's booking engine in the background.  
- **Hosting approach:** The plan is to build a custom site (not Wix/Squarespace) hosted on a low-cost platform like Netlify (free tier available). This keeps annual costs near just the domain (\~$15/year) rather than $200–400+/year for a website builder subscription.  
- **Who maintains it:** Jay will handle technical updates; Bethany just needs to be able to tell him what to change.
